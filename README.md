# 🥁 Drum Kit

A simple web-based Drum Kit built using HTML, CSS, and JavaScript. Users can play different drum sounds by either clicking the drum buttons or pressing the corresponding keys on the keyboard.

---

## 📸 Preview

![Drum Kit](Screenshot1.png)
---

## 🚀 Features

* Play drum sounds using keyboard keys
* Play drum sounds using mouse clicks
* Interactive button animations
* Real-time audio playback
* Responsive and engaging interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Drum-Kit.git
```

2. Open the project folder

3. Run `index.html` in your browser

4. Click the drum buttons or press the corresponding keys on your keyboard
